# GeoCompanion.ai Pitch Deck (Standalone)

This is a separate React/Vite project for the pitch deck.

## Run locally

```bash
cd /Users/devacct/coding/pitch_deck
npm install
npm run dev
```

Open `http://localhost:5173`.

## Export

Use the **Download PDF** button in the top bar (or browser print) to save the deck as PDF.
